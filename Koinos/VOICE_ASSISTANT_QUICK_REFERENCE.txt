 🏗️ ARCHITECTURE
 ═════════════════════════════════════════════════════════════════════════════

 VoiceAssistantView
         ↓
    User sends message
         ↓
 VoiceAssistantViewModel.sendMessage()
         ↓
 KoinosVoiceRepository.processVoiceCommand()
         ↓
 Get Firebase Token from AuthRepository
         ↓
 POST to http://localhost:8000/api/voice/
    Header: Authorization: Bearer <token>
    Body: { "message": "..." }
         ↓
 FastAPI Backend processes with LangGraph agent
         ↓
 Backend returns JSON: { "message": "AI response" }
         ↓
 ViewModel receives response
         ↓
 Add message to conversation
         ↓
 VoiceAssistantView updates UI


 🔐 SECURITY FLOW
 ═════════════════════════════════════════════════════════════════════════════

 1. User sends message through VoiceAssistantView
 2. ViewModel calls repository
 3. Repository gets current Firebase ID token
 4. Token is automatically injected into Bearer header
 5. HTTP request sent with token
 6. Backend receives request with token
 7. Backend verifies token with Firebase Admin SDK
 8. Backend extracts user_id from token
 9. Backend processes with verified user identity
 10. Response returned to iOS app
 11. Message displayed in chat


 📱 USER JOURNEY
 ═════════════════════════════════════════════════════════════════════════════

 1. User launches app
 2. User logs in (Firebase auth)
 3. User navigates to "Assistant" tab
 4. Koinos AI welcome message appears
 5. User types message in input field
 6. User taps send button (paperplane icon)
 7. "Processing..." indicator appears
 8. User's message shows in blue bubble (right side)
 9. AI response appears in gray bubble (left side)
 10. User can continue conversation
 11. User can tap refresh to clear conversation
 12. User can tap sign out to return to login


 💬 MESSAGE TYPES
 ═════════════════════════════════════════════════════════════════════════════

 User Messages:
 ├─ Blue background
 ├─ Right-aligned
 ├─ Example: "Add an expense for $50"
 └─ Sent with: isUserMessage = true

 AI Messages:
 ├─ Gray background
 ├─ Left-aligned
 ├─ Example: "I've added your $50 expense"
 └─ Sent with: isUserMessage = false

 Error Messages:
 ├─ Gray background (like AI)
 ├─ Left-aligned
 ├─ Prefixed with "Error: "
 └─ Also shown in red banner at top


 ⚙️ STATES & INDICATORS
 ═════════════════════════════════════════════════════════════════════════════

 While processing:
 ├─ "Processing..." text appears in chat
 ├─ ProgressView spinner shows
 ├─ Send button is disabled
 ├─ Input field is disabled
 └─ isProcessing = true

 When ready:
 ├─ Send button enabled (if input not empty)
 ├─ Input field enabled
 ├─ No spinner shown
 └─ isProcessing = false

 Error state:
 ├─ Red banner at top shows error message
 ├─ Error message also added to chat
 ├─ User can dismiss banner with X button
 └─ errorMessage property set


 🔧 CONFIGURATION
 ═════════════════════════════════════════════════════════════════════════════

 Backend URL:

 - Endpoint: /api/voice/
 - Method: POST
 - Auth: Bearer token in header

 Token source:
 - Firebase Auth iOS SDK

 Request format:

 }

 Response format:
 {
   "message": "AI's response text"

 💡 DEBUGGING TIPS
 ═════════════════════════════════════════════════════════════════════════════

 No response appearing:
 ├─ Check backend is running
 ├─ Check Firebase token is valid
 ├─ Look for errors in console
 └─ Verify URL is correct: http://localhost:8000/api/voice/

 Unauthorized error:
 ├─ Check Firebase Admin SDK initialized on backend
 ├─ Verify token is being sent in Authorization header
 ├─ Check token is not expired
 └─ Re-login to refresh token

 Network error:
 ├─ Ping backend: ping localhost
 ├─ Check firewall settings
 ├─ Verify network connectivity
 └─ Check localhost:8000 is accessible

 Message not sending:
 ├─ Check input field is not empty
 ├─ Check isProcessing is false
 ├─ Try previous message again
 └─ Check for errors in ViewModel

 UI not updating:
 ├─ Verify @Observable decorator
 ├─ Check @MainActor on sendMessage()
 ├─ Ensure state properties are var
 └─ Try hot reload
