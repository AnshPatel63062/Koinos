 📱 USER EXPERIENCE FLOW
 ════════════════════════════════════════════════════════════════════════════

 Step 1: User Opens App
 └─ Logs in with Firebase
    └─ Authenticated with ID token

 Step 2: Navigates to Expenses Tab
 └─ Taps "Expenses" (creditcard.fill icon)
    └─ TabView shows ExpenseListView

 Step 3: List Loads Automatically
 └─ ExpenseListView appears
    └─ loadInitialData() called
       └─ fetchExpenses() called
          └─ GET /api/expenses/ with Bearer token
             └─ List displays all expenses

 Step 4: View Expense Details
 └─ User taps an expense
    └─ NavigationLink navigates to ExpenseDetailView
       └─ Shows all details in view mode
          ├─ Amount (green, currency formatted)
          ├─ Category (with icon)
          ├─ Date (formatted)
          └─ Description (if present)

 Step 5: Edit Expense
 └─ User taps "Edit" button
    └─ Switches to edit mode
       ├─ TextField for amount
       ├─ Picker for category
       └─ TextEditor for description

 Step 6: Save Changes
 └─ User taps "Save"
    └─ Validates: amount > 0
       └─ PUT /api/expenses/{id} with Bearer token
          └─ Backend updates expense
             └─ Response updates detail view
                └─ Returns to view mode

 Step 7: Refresh List
 └─ Pull down in list view
    └─ Animated refresh indicator
       └─ refreshExpenses() called
          └─ GET /api/expenses/ with Bearer token
             └─ List updates

 Step 8: Delete Expense
 └─ Swipe left in list view
    └─ Delete action appears
       └─ Tap delete
          └─ deleteExpense() called
             └─ DELETE /api/expenses/{id} with Bearer token
                └─ Item removed from list
